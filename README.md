# TaskTracker

Автор: климова Дарья 25ИС11Д

## Описание
Учебный проект: трекер задач (консольное приложение).

## План функций
- Добавить задачу
- Показать список задач
- Изменить статус задачи
- Удалить задачу

## Стек
C# / .NET / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
3) В консоли появится меню
