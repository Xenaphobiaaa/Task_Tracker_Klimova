\# Отчёт ЛР5 — TaskTracker



\## Что сделано

\- В модель TaskItem добавлено поле Description

\- В TaskService добавлен метод Update(id, newTitle, newDescription)

\- В консольное меню добавлен пункт 5) Редактировать задачу

\- После редактирования данные сохраняются в data/tasks.json



\## Как проверить

1\) Запустить (F5)

2\) Добавить задачу

3\) Редактировать задачу: поменять Title и Description

4\) Показать список — видно описание

5\) Перезапустить программу — изменения сохранились
