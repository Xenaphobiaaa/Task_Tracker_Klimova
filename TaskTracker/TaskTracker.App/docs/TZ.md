\# Мини-ТЗ: TaskTracker



\## 1. Название

TaskTracker (Трекер задач)



\## 2. Цель

Программа помогает хранить список задач и менять их статус.



\## 3. Пользователь

Обычный пользователь ПК (студент).



\## 4. Функциональные требования (FR)

\- FR1: Добавить задачу (название обязательно)

\- FR2: Показать список задач

\- FR3: Изменить статус задачи (Новая / В работе / Готово)

\- FR4: Удалить задачу



\## 5. Нефункциональные требования (NFR)

\- Приложение запускается без ошибок

\- Понятные сообщения об ошибках (например: “Пустое название нельзя”)

\- Данные будут храниться локально (позже: JSON-файл)



\## 6. Модель данных (черновик)

Задача Task

\- Id (число)

\- Title (строка, не пустая)

\- Status (New / InProgress / Done)



\## 7. Критерии приемки (как проверить)

\- Можно добавить 3 задачи и увидеть список

\- Можно поменять статус 1 задачи

\- Программа не падает при неправильном вводе



\## 8. План модулей (черновик)

\- Core (модели и логика)

\- App (консольное меню)

\- Storage (позже: сохранение/загрузка)
